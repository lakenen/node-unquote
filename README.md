# unquote

Remove wrapping quotes from a string.

## Installation

```
npm install unquote
```


## Usage

Example
```js
var unquote = require('unquote')

unquote('"hello, world"') // 'hello, world'
unquote('\'hello, world\'') // 'hello, world'
```
